Calculator Functions & Features

Basic Functions

  • Addition: Use the + key to add numbers. Example: 2 + 3 = 5
  • Subtraction: Use the - key to subtract numbers. Example: 5 - 2 = 3
  • Multiplication: Use the × key to multiply numbers. Example: 4 × 5 = 20
  • Division: Use the ÷ key to divide numbers. Example: 10 ÷ 2 = 5
  • Modulus: Use the % key to find the remainder after division. Example: 5 % 2 = 1

Advanced Functions

  • Square: Use x^2 to square a number. Example: 5^2 = 25
  • Power: Use x^y to raise a number to any power. Example: 2^3 = 8
  • Square Root: Use 2√y to find the square root. Example: 2√9 = 3
  • Nth Root: Use x√y to find any root. Example: 3√27 = 3
  • Logarithm: Use log_x(y) to calculate logarithm with base x. Example: log_10(100) = 2
  • Euler's Number: Use e to represent Euler's number (approximately 2.71828).

Trigonometric Functions

  • Sine: Use sin(x) to calculate the sine of an angle in degrees. Example: sin(90) = 1
  • Cosine: Use cos(x) to calculate the cosine of an angle in degrees. Example: cos(0) = 1
  • Tangent: Use tan(x) to calculate the tangent of an angle in degrees. Example: tan(45) = 1

Function Plotting

In the Trigonometric Calculator section, you can plot mathematical functions:

  • Enter a function using the variable x (e.g., sin(x), x^2, 2*x+1)
  • Set the range for the x-axis using the range controls
  • Click the "Plot" button to visualize your function
  • The graph will update automatically to display your function

Setting Decimal Accuracy

You can set the number of decimal places displayed in your results:

  • Enter a number (1-10) in the "Decimal Accuracy" field
  • The calculator will round results to that many decimal places
  • Default accuracy is 2 decimal places
